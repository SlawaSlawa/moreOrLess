# moreOrLess
Игра больше-меньше
## Описание:
Компьютер случайным образом загадывает число, Ваша задача его отгадать. Также записывается рекорд (localStorage) с возможностью его сбрасывать
### В разработке были задействованы
- HTML
- CSS
- JS
